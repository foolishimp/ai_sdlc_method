# genesis_manager

**A builder supervision console for Genesis.**

genesis_manager is the control surface for a person using Genesis to build projects. It answers the question: "What is Genesis building for me, and what should I do next?"

---

## Quick Start

```bash
# Bootstrap session with workspace state and NL routing
/gen-genesis

# Check project status
/gen-status

# Start or continue work
/gen-start
```
